          # 斯蒂格勒《技术与时间3》引言 //

          意识具有时间性，它是和音乐旋律一样不停流逝、消失的瞬间，但它还处于演变状态的历史性中，因此它不是一个永恒物，而是一个不断进行综合的过程。

          综合能力是意识在面对记忆的支撑物，也即第三持存，进行感知时（由此对象变为了「时间客体」）将产生的感知进行统一的能力——首先产生了新的感知，然后综合为新的记忆，构成统一的意识流。

          但随着超工业化社会中信息技术、远程通讯技术、视听技术的发达，工业时间客体（如电影、电视、手机）使得意识流能够使自身统一的能力完全被短路了。

          如果说技术物是一种体外化器官，是对人体身心器官功能的延伸与外化，那么可以说，我们今天进入了一个意识及其综合能力被代具化的时代，所有持存机制被系统性地工业化所接替。意识及其综合能力的外包，导致了我们不再能够使自身心理个体化。

          而个体化的丧失便是我们所说的无产阶级化——也即西蒙东所描述的「技术个体」（技术物）对人的取代。个体的意识——简言之，注意力——在今天除非与“世界”相割离，否则便注定会沉溺于程序工业的“巨流”之中无法自拔。大数据算法所谓的“个性化”，接替、解体了个人真正的独特性（那一专属于ta的知识、欲望、行动、精神）。

          # 斯蒂格勒《技术与时间3》9-10 //

          故事欲——也即一种「叙述/说」的欲望——是文化的基础，因此它几乎伴随了人类的历史。故事是将过去投注到未来，这种投射的欲望，或者说癖好和痴迷，代代相传并成为维系着各代人之间的纽带。

          而今，满足这种投射欲望的条件已经发生了剧变，它已成为世界级工业活动的对象——「文化工业」构成世界经济发展的核心，今天的故事欲高度臣服于传输工业的发展。

          视听技术、信息技术、电子通讯技术互相结合，使得故事叙述的形式具备了一种极其特殊且前所未有的力量，激发了人们对故事的信仰。

          因此，我们应当分析与电影一同出现的各种技术之特殊性，这些技术作为了一种控制式的“程序工业”具有无比的高效性，构成了全球工业所生产的象征物与传播技术网络。我们将分析例如动态有声图像为何以及如何能够满足了普遍存在的故事欲，分析电影为何以及如何冒着穷尽人类故事欲的危险调节了整个人类的演变过程，以及我们将会阐释电影之特殊性如何揭示了“人类灵魂”之特殊性——意识犹如电影。

          # 斯蒂格勒《技术与时间3》11-12 //

          一个秋天周日郁郁寡欢的下午，正如诸多类似下午中的其中之一，一个什么都不想做，因为他不知道该做什么的倍感烦闷之人，他没有理由不去打开电视或进入一部电影。

          他为什么不拿起一本书，比如一本故事情节写得相当动人或过瘾的书？为什么与词句相比，图像占尽上风？这是因为，在银幕面前，他什么都不需要做，只需慵懒地睁眼观看即可，其中的内容并不重要，重要的是，电影/电视作为动态有声图像的时间流，代替了他的意识流进行运动，坐在沙发上的他，感觉自己重新地“活了过来”。动态有声图像之运动的流逝，代替他活了生命中的90分钟或52分钟。在里面，它不太需要观众进行干预、参与与介入，不需要观众聚精会神地担心遗失了文本的主线。

          电影的功能是针对我们的烦恼，它使我们遗忘或转移了该烦恼的实体，因此我们若有幸看到部还不错的片子，电影会使我们重新拥有对事物的期待，它是对“现实”的重新剪辑。但是，这种“剪辑”以及种种对“图像”的处理能力，是我们人的欲望本身就具备的能力，换言之，今天的电影或整个文化工业成为了我们意识、欲望的体外化器官，它来代替、接管乃至扭曲、短路掉我们的这一原始能力了。

          # 斯蒂格勒《技术与时间3》12-4 //

          电影的两大根本原则：

          1.摄影是一种复刻性的记录。复刻性是指摄影具有现实性的效果。巴特认为，照片的意象内容（noème）便是“曾经存在”：绘画可以虚构一个现实物而不必真的见过它，但摄影却必然地指涉一个现实性的事物。对于摄影而言，它是“现实性”与“过去（曾在）”的重叠。看一副照片时，照片像接收者提供了这样一个时刻——在其中某一现实事物在镜头前处于静止状态，而接收者则会通过照片回溯到过去被拍摄的这一刻，也即“曝光”的一刻，因为照片本质上就是这一刻。换言之，照片像是摄魂一样，使接收者回到了某个曾经存在过的现实时刻，接收者的意识与那一时刻重叠了。

          而在“现实性”与“过去”相互结合的意义上，电影正是摄影的一种延伸。

          2.录音与摄影一样都是一种人工复刻性的记忆技术，它是关于声音的曾经存在、已经发生。但不同的关键之处是在于，音乐或声音是一个流动的客体，是一个时间流，接收者无法仅仅抓取它某个瞬间，音乐客体不存在类似于曝光的时刻，而是胡塞尔所言的「时间客体」。

          有声电影出现后，电影将声音也纳入其记录之下。电影不再仅是图像记录技术，而是重现运动，我们便可以将胡塞尔对时间客体的现象学分析带到电影分析当中。

          我们想要强调的是，基于电影所具备的上述两大原则，使它作为了一种具备过去之现实性的「流」，观众的意识流将会与电影这一时间客体的时间流重叠。

          过去之现实性促成了一种“真实效应”，使人信服，而这种信服使得电影可以提前确定观众的姿态（它预先制造好了观众的欢笑、悲伤、震撼、感动）。

          如此，电影总是对意识到接管，我们可以说它是对意识活动的“疏通”（例如心情不好时通过看电影来“换换心情”），又或不幸地是对意识活动的捕捉、塑型。

          # 斯蒂格勒《技术与时间3》14-5 //

          德勒兹认为电影是两种材料之间的互补：一是图像，二是一种作为了无人称、均匀、抽象、不可见的机械式时间的运动，这一运动使图像得以前后相连。

          因此，电影实际上给予了我们一种虚假运动，柏格森为这一虚假起了一个现代新潮的名字：“电影的幻觉”，可是德勒兹反驳于此，认为这种幻觉其实是古老、永恒而普遍的——人的意识活动从来都是这样的电影机制，也即一种将图像统合为运动的能力，并不像柏格森认为的那样，是现代电影的出现才带来了这种虚假运动的幻觉。

          德勒兹认为，对幻觉的复制同时也是以某种方式对幻觉的纠正，当意识本身已就是某种虚构（不能作为被敲定的恒定“客观现实”），我们便需要重新主动地动用自身的综合能力，去给出运动、时间，乃至互相交换这样一种“给出”，而不是依附、信仰于某一陈套。要用魔法来打败魔法。

          但德勒兹并未从其观点出发来推出所有会造成的结果，因为他没有考察图像作为了一种“曾经存在”的复刻性，因此无法分析是什么使活动图像具有了巨大的力量。

          换言之，我们想提出的是，记忆的“曾经存在”被保存为了物质性的第三持存，我们给出运动、时间的能力，也即我们的意识活动，正建基于这些已然存在的作为体外化器官的技术物。

          # 斯蒂格勒《技术与时间3》16-9 //

          时间客体不仅仅是单纯地存在于时间中，而是必须在时间之流中得以自我构成、自我组织，它不是很恒定或永恒之物，而是一个不断一边呈现一边消失的过程，是一个在产生过程中消逝的流。

          当胡塞尔从静态现象学转向发生现象学时，正是因为他发现意识之意向性不可能单纯以某事物为对象，而只有在意识对象作为时间性客体时，对意识的时间性分析才可能实现。

          意识本身就是一个流，其对象也是流。为了分析意识流的时间质料，分析时间客体便是一个很好的对象，因为它的时间流完全与以之为对象的意识流相重合。分析时间客体的时间流，也就是分析以之为对象的意识流了。

          以音乐旋律为例，第一持存便是对旋律每一瞬间的流逝过程统合起来的印象，也即把所以刚刚已经过去了的“此刻”，持留保存住，并融合进下一“此刻”之当下中。虽然音符的每一瞬间都在流逝，但它们被持留在了当下这一“此刻”之中，因此它们既不在场又在场。只要时间客体没有完全流逝、过去、中断，这一过程便一直如此，只有这样所有瞬间的音符才能被感知为具有统一性的音乐旋律，才成为了一个时间客体。

          简言之，第一持存（或第一记忆）便是把“此刻”与“刚刚过去的时刻”联系在一起，使之保持同时在场，时间客体也由此能够呈现为一个自我构成、自我组织的对象。“刚刚过去的时刻”被保存为了一种“前摄”。

          再以电影剪辑中的“库里肖夫效应”为例。在库里肖夫实验中，一个演员脸部特写镜头被插入到不同的段落之中，这一完全相同的镜头却由此在观众意识中产生了不同的“表情”含义。

          这便是意识里的电影机制或称蒙太奇机制——意识会将记忆或印象投射到当前的对象之上。意识和电影都在进行着这样的操作：对若干图像进行组装，使之成为一个统一的时间流。

          布伦塔诺（胡塞尔、海德格尔以及弗洛伊德的老师）并未区分感知与想象，他将“刚刚过去的时刻”也视为了已是一种想象，已经是模糊的、消逝的不在场了。胡塞尔反驳于这一点，认为布伦塔诺这一观点实际上是在声称对时间客体的感知是不可能的，所有的感知已是想象的结果，换言之，这个观点完全否认了时间本身。

          胡塞尔由此区分了第一持存和第二持存，后者才是想象活动，例如是在第二天回想起昨晚音乐会旋律的记忆，中间存在着时间客体之流的中断，它与第一持存的感知活动将时间客体组织为统一对象的过程是不一样的。

          因此，胡塞尔将第一记忆与第二记忆对立起来，认为感知和想象之间毫无瓜葛，被感知之物绝非被想象之物，前者是非虚构的直接在场。

          但是胡塞尔因此也拒绝了生活和电影（或思想）的联系，将两者做出绝对的区分，胡塞尔想要关注的是作为感知着直接在场的生活世界，而悬置掉其它虚构性，他认为生活绝非故事性的。

          # 斯蒂格勒《技术与时间3》19-24 //

          我们认同于胡塞尔对第一持存和第二持存的严格区分，但却不认同于将两者视为对立。事实上，这一将它们视为对立削弱了这一区分的内涵。我们将提出，感知从来都是不纯粹的，总是含有想象的成分，由此任何感知都是一种投映。将区分与对立混淆，或许是一种形而上学传统所惯有的混淆。

          感知是想象得以投映的银幕，因此，我们或许就可以说生活总是电影。为此，我们提出了第三种记忆、第三种持存，它是记忆得以“复活”——重新找回一种生活——的前提，也是构成了 前摄 的载体——它构成了期待，激发了感知与想象。

          胡塞尔认为感知是纯粹的，但实际上意识在抓住流逝之物过程中进行了遴选，那么它抓住的其实并不是纯粹的感知，而已构成某种想象。

          这也是为何我们听同一段音乐两遍时，会感知到不同的听觉现象，这正是由于遴选过程发生了变化。意识不仅不会抓住所有一切细节，而且每次抓住的部分都不一样。

          而我们想要强调的是，正是由于模拟/数字第三持存的诞生（如唱片、音频），意识过程中对第一持存的遴选，也即想象对感知的介入这一事实才变得显而易见，因为这样的第三持存保证了时间客体几乎可被完全同质化地重复播放，由此每次感知的不同现象，得以明确地被归结为是想象的结果。

          我们遴选过程，正是三种记忆之间互动的过程：作为第三持存的技术物同时激活了另外两种持存，而作为第二持存的想象或记忆则通过蒙太奇，为作为第一持存的感知提供着“上下文”，换言之，想象介入了感知（正如库里肖夫效应）。

          每次感知的过程都是基于由前摄所构成的期待而进行遴选，因此带着不同的记忆就会有着不同的遴选准则，得到不同的感知。遴选实际上意味着，我们记忆、保存的过程，同时也是遗忘、遗矢的过程。但遴选是必要的，因为假如我们能够100%记住并还原昨天24小时内的每一秒，那么便不会带来任何差异，我们第二次进入同一首音乐时便不会感知到不同的现象。正是由于遴选是一种简化，作为“已经听过之物”才得以促生了“尚未听过之物”，遗忘也是为了更好地感知。流逝过程中的持存是对它自身的更正，流逝使得熟悉之物神奇地给了我一些陌生之物。记忆（无论是第一或第二记忆）的磨损为我们带来一次次不同的遴选，将正在流逝之物简化为过去时刻，后者提供着期待。任何回忆都是一种遗忘，如果没有了这一遗忘，时间便不可能发生。

          # 斯蒂格勒《技术与时间3》26-9 //

          电影角色的生命和演员自己的生命重叠了在一起。当一个演员年老后观看自己曾经出演过的电影时，她正在经历的，是一种对自身正在死去的体验，换言之，她既体验着曾经自己以及演出过角色的生命所具备的年轻与美貌，又体验着这种美貌与生命的已经死亡。所有的照片、影片都是这种灾难，它所述说的都是关于死亡的先将来时。

          # 斯蒂格勒《技术与时间3》54-6 //

          “三重综合”指的是康德所说的“知性之自发性”的运作过程。

          知识是一个由互相联系的杂多表象所组成的整体，任何一种综合总是某种“概述”（synopsis）。

          知性之自发性是三重综合的原则，三种综合是知识的三种主观来源，正是它们使知性成为可能，并通过知性使一切经验得以可能。

          这三种知识分别是：

          1.对直观中思维的各种变化表象进行的领会综合（联系于第一持存，也即感知）；

          2.这些表象在想象中的再现综合（联系于第二持存，也即回忆或想象）；

          3.这些表象在概念中的认定综合（联系于第三持存，也即经验/记忆之书写化的储存）。

          换言之，三种知识分别是对杂多表象——在感知中的包扎、在想象（记忆）中的包扎、在概念（也即以外置化的书写方式）中的包扎。这个包扎/打包/综合的过程，便是知性之自发性，也即知性总是会自动地对感性材料（表象）进行这三种操作，也只有这样经验和知识的形成才得以可能。

          三重综合中，尤其领会综合，是一个时间问题：直观杂多之所以需要被整理，是因为我们接收到的所有表象都受制于内感官——所有表象都必须在时间内得到整理、结合并互相建立关系。

          而再现综合，则是在表象互相联结、形成关联后，以至于即使对象不在场了，也能根据想象或回忆，在思维中再现这些联结（例如看到苹果时就联想到它的味道，即使这时还没尝到）。

          然而，康德最终却将两种综合能力（换言之，也即第一、二持存）混为一谈，他提出，再现综合其实就是领会本身之中的持留，也即想象支撑了感知，感知本身内就包含了想象。这种混淆也正是布伦塔诺同样采取了的角度，而胡塞尔则对之进行过指责。

          康德实际上谈论的只是第一持存，但他却以为自己在描述第二持存，认为是想象使感知得以可以，康德由此总结道：领会综合与再现综合不可分割地联结在一起。换言之，康德并不认为第二持存是作为遴选准则伴随着第一持存，而是声称领会从一开始就是再现，领会已等于再现。这里的“再现”之定义是明确的，也即第二持存（回忆/想象）。这也是为何康德提出了不可知的物自体——不存在对事物本质的直接感知，因为我们对其第一时间的感知就已经是再现（回忆/想象）后的表象了。

          但我们认为，两者必须进行区分，感知必须区别于想象，两者并不是一回事，但却并不对立——想象是作为了遴选准则作用于了感知，两者确实交织在一起，但感知仍不是想象。

          总言之，领会综合（第一持存，感知）是对“刚刚过去的时刻”在当下时刻中的存留，以及作为了“尚未到来的时刻”的前摄。如果康德意识到这点，那么或许他便能区分开两种综合。

          # 斯蒂格勒《技术与时间3》57-8 //

          三重综合中的最后一种，也即认定综合，是在构成某对象的概念之统一性，知识是必须与某对象之间产生联系并互相协调一致时才会形成（赋予概念）。

          认定综合确保了意识作为一种流的统一性，使它们作为一个整体而不相互抵触。意识之流的统一性，则决定了领会综合和再现综合之间的统一性，使两者统一为认定综合。换言之，感知与回忆（前两种综合或持存）使得某对象出现在意识中，而认定综合则是对所有感知和记忆在意识流中的统一，这也使对象本身在我们意识中得到了统一（例如我们会把苹果的颜色、口感、味道等等分散的感觉或记忆，统一为“一个苹果”这个概念上，而非分散的杂多感性材料）。

          这种意识在其对象中的统一性，其实是统觉的外在表现。而康德认为这种统觉是先验的，它体现了概念（规则）之先天必然的必要性——直观中这种先于经验的统合功能，按照一定规则，才使对表象的经验以及将杂多表象结合于一个概念中得以可能。先验的统觉使得统一性发生。因此在康德看来，意识的先验统一性，便是对象乃至宽泛意义上整个外在世界的先验统一性（这便是康德所说的先验亲缘性，意识和表象世界之间的亲缘性）。

          概念对经验中杂多的再现之物进行统一，这种统一同时也即是“再-生产”，但从更深层次来说“再-生产”就是“生产”，因为是概念使一切经验流、时间流得以可能。没有概念的统一性，经验与时间都变得不可能，所以，说概念“再-生产”（再现、复制）了经验，不如说实际上是“生产”（创造）着经验。

          认定综合（意识流的统一化）不仅将已经过去的杂多经验（可再现、可回忆的过去时刻）进行统一，同时也为即将到来的时间流以及在这些流中构建起的对象之统一化做好了准备，换言之，认定综合作为了一种经验的前摄，它以过去的经验之统一为前提，并将其带到了即将到来的感知当中去，使它们能够有机地统一为整体。但同时，这所借助的正是对感性杂多的领会综合，认定综合的统一化才得以可能。三种综合正由此变得交织在一起不可分割，而三种综合是三种持存的另一种表达方式，后者同样也是必然地联结在一起而不可分割。这就是我们接下来将要考察的对象。

          # 斯蒂格勒《技术与时间3》62-5 //

          康德认为统觉之统一性是先验的，所以只要严格地动用理性，那么每一次都能够将某种合理性给辨识出来并固定下来，能够一次次地还原出先验统觉之统一性，并将所有未来之感知或曾经之记忆中的多样性都给统合进去。

          海德格尔强调了“已经在此”的世界历史性与此在的在世性之间的重要关联，但却忽略了正是对作为载体的第三持存中的时间进行继承，才打开了“世界”。这一继承，也即传输和接受，或保存和教育，海德格尔也由此忽略了教育问题以及电影机制的接受问题——第三持存正是我们集体地打开一个世界之“已经在此”的屏幕，也是对未来进行投映的前摄。

          第三持存既是时间的空间化（例如思想被记录为了文本），也是空间的时间化（例如文本作为屏幕对历史的打开和未来的投映）。第三持存是进入“已经在此”的前提条件，正如海德格尔在《存在与时间》已明确指出的那样，“已经在此”正是继承之各种可能性的实现，构成了此在的历史（以及世界）。

          我们试图区别于海德格尔而强调，本真时间与非本真的计算性时间（正如共时性和历时性）是无法泾渭分明地进行对立的，我们必须依赖和继承于第三持存，基于非本真的共时性，随后才能进入此在之本真性。正如知识的传输必须被简化为压缩包（文本），而之后才能在教育以及接受的过程中打开每个人分枝的“本真性时间”。海德格尔拒绝承认具有电影特性的接受以及由接受所构成的诸多演变进程。

          # 斯蒂格勒《技术与时间3》232-3 //

          技术物对于人而言的复杂性在于，它既是人的内在化也是人的外在化，而且内-外在化是同一时间而非相继发生的：内在化就是外在化，个体化的过程，既将独特性外在化（或者说分枝）为物质痕迹保存了下来，但也正是在这一过程中才构成了个体——生命是一种同时激活了“谁？”与“什么”的活动——“我”之个体性以及“已经在此”之原则，同时得以构成。

          我们可以发现，所谓的“主客体”并不存在二元式的等级制关系，而是同时形成、互相传导的关系，海德格尔所说的“世界”或我们称之为“时代” 正是这一过程中打开的。同时，我们还需要强调，正是由于这样一种世界或时代之历史性，“谁”的后面才必须有一个疑问号或省略号而非句号，以及，从不可穷尽的“真实”落实为了“什么”之形式化的原则，虽然是不可避免的，但二者之间的深渊也是不可缩减的。

          简言之，“谁？”与“什么”之间的传导关系，永远将是一种未终结的延迟与差异的关系，也即处于一种亚稳态当中。

          # 斯蒂格勒《技术与时间3》235 //

          意愿（或欲望）是控制的对立面。当技术力量转变为了资本主义工业与科技之时，尼采提出了对力量的拷问——“我们”想要（意愿、欲望）的是什么？而这一问题，也即欲望的问题，正是对引号中的“我们”进行的质疑——在科技的力量背后，存在着一个能够称之为“我们”，也即一个集体对欲望进行未来投映的共同体吗？

          我们把“不知道想要什么却必须做出使用着力量的人”称为“无科学的人”，一个丧失了欲望而被力量控制着的人，这是一个关于存在的问题，是存在之不适乃至躁郁的问题。

          # 斯蒂格勒《技术与时间3》242-3 //

          理性是这样一种欲望乃至信念——进行判断。理性对经验进行判断，即便在完全没有任何可感知的客观数据或现实数据支持的情况下，仍要为该经验提供一整套完整的依据，也即从起源直到终结——理性是一台对经验之一切可能性进行投映的电影放映机。

          换言之，理性必须是一种缺失，它必须是非完满的，以至于像一头发情的牛寻求着进行判断的对象，并对自身的判断保持欲望和信念。与其说理性具有这一欲望，毋宁说理性本身就是这一欲望。理性是一种对知识的爱（欲望），而其背后必要的动力机制，便是缺失——它是必须的，我们需要它，必须信任它，因为它正是理性的动机，使理性处于运动之中。

          理性背后所投映的依据，便是上帝，作为超感知之现实的一切可能性之全部的根据。这是理性的一个需求，也即理性需要认为存在着一个一切可能性的绝对依据，其它一切都是这一初始原则的支流。

          # 斯蒂格勒《技术与时间3》252-3 //

          在亚里士多德看来，科学是关于客观性、必然性和确然知识（savoir apodictique），而与之对立，技术则是关于偶然性的、“无科学”的技能（savoir-faire inscient）。技术是具有历史、可演变的，科学则是形式化的。这也是为什么“Tekhnen tukhen esterxe kai tukhen tekhnen”——技术爱着偶然，偶然爱着技术。

          我们应当从“将技术视为对科学的某种运用”这一现代思维模式中解放出来。后者正是自笛卡尔-伽利略到康德-牛顿的“现代科学”观念，认为现代科学遵循于大自然中的因果关系，而这样的因果关系之多，给偶然性留下了一定空间，也即给人类活动留下了一定空间——科学确定客观、必然的因果规律，而技术是对这些规律的运用。

          然而，我们必须还将“现代科学”与“当代科学”进行区分。如今，科学和技术之对立才被真正地打破而混合为了“科技”。科技一般会被视为基于各个技术手段的应用科学，我们则在此提出一个颠覆性的观点：不是技术变成了应用科学，而是科学变成了应用技术。

          当代科学产生了一些形式化的成果，而这些成果已变得可被复制，也即可通过自动化机制而被复制，当代科学由此成就了一种具有自动复制性的特殊世界，在这个世界中，科技（技术科学）根本不满足于遵循各种“客观的”因果关系，而是制造出因果关系，正如试图控制河流来开发水力，又或今天生物学基因工程在做的事情一样。当代科学在制造法则，以便成为一种应用技术。换言之，当代科技既无关偶然性，也不是对自然因果规律的客观观察，它是一种极为特殊的持存，对其制造的因果关系法则的持存（工业化复制）。

          # 斯蒂格勒《技术与时间3》255 //

          当代科技的出现导致了现实性（自然客观）退居二线，对可能性进行系统地研发则构成了市场投资的首要目标。投资（资本流的方向）变成了一种预先准备，它决定了哪些可能性得以被实现。

          技术、科学、资本流三者之间的联合，导致了任何所谓“客观现实”已从属于过去，一个通过实验而得以系统地研发的未来拉开了序幕——资本与科技的逻辑现已成为我们的现实之将来的法则。
